## 📁 项目结构

```
it-community/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
```

## 🔧 使用说明

### 本地运行

1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (需要安装 http-server)
   npx http-server -p 8000
   ```

3. 访问 `http://localhost:8000`

### 自定义内容

- **修改配色**：编辑 `css/style.css` 中的 CSS 变量
- **添加话题**：在 `index.html` 的 `.topics-grid` 中添加卡片
- **添加文章**：在 `index.html` 的 `.articles-list` 中添加文章
- **添加项目**：在 `index.html` 的 `.projects-grid` 中添加项目
